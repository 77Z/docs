V Strap Docs

This is the offical docs for the API: V Strap. This documentation includes everything you nned to know about V Strap. This page was built with V Strap too.

Installation

Very Simple, there is a regular version and a minified version. They stay up to date with each new major release.

v-strap.css
v-strap.1.0.0.min.css

Make sure you have the font: Google Sans placed in a folder by the V Strap file named "google-sans" Disincluding the quotes( "" )

Colors

Every element you place on the page can have it's background and text color changed by using class='bg-COLORNAME' OR class='color-COLORNAME'. The different colors we have are: red, pink, purple, indigo, blue, cyan, teal, green, lgreen, yellow, amber, orange, dorange, black, and white.

The default background color is blue; the default text color is white.

Buttons

To make a button, use an <a> tag with a class of: btn. If you want to make a raised button, use the class: btn-raised

Like I say in the Colors Section, You can change the text color and the background color the way the Colors Section Explains

Example: <a href="#!" class="btn">Button</a>

Button

Example 2: <a href="#!" class="btn-raised">Button</a>

Top Bar

The top bar is used in this site, as you can see, the bar at the top reads: "V Strap Docs"

To create this, use a <div> tag, with a class of top-bar

If you put a <span> tag inside of the div, then the span will act as the title and enlarge to fit the bar.

Example:
<div class="top-bar">
    <span>
        My Title
    </span>
</div>
